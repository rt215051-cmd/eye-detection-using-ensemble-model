# eye-detection-using-ensemble-model
Technologies used in this project include Python, OpenCV, TensorFlow, Keras, NumPy, and deep learning frameworks. The project demonstrates how artificial intelligence and ensemble learning techniques can improve the performance of eye detection systems and support the development of intelligent safety and monitoring applications.
Description:

The Eye Detection Using Ensemble Model project is a computer vision and deep learning system designed to detect human eyes accurately from images or live webcam video. The system combines multiple machine learning or deep learning models together using an ensemble approach to improve detection accuracy, robustness, and real-time performance.

The project mainly uses OpenCV for image processing and CNN-based deep learning models for feature extraction and classification. Ensemble learning combines predictions from multiple classifiers such as CNN, Haar Cascade, ResNet, MobileNet, or other pretrained models to achieve better results than a single model.

The system performs the following operations:

Captures image or video input
Detects face region
Identifies eye regions
Classifies eyes as open or closed
Improves prediction accuracy using ensemble techniques
Displays detection results in real time

Main Features:

Real-time eye detection
Open/closed eye classification
Webcam support
High accuracy using ensemble learning
Deep learning based prediction
OpenCV integration
Drowsiness and attention monitoring support
